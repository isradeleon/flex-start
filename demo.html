This demo showcases the differences between justify-content, align-items, align-content, align-self, and shorthand properties like place-content.

justify-content (Main Axis Alignment)

Aligns items along the main axis (the direction of flex-direction). In flex-direction: row, this is horizontal. It controls spacing and positioning of items within each flex line.

justify-items (Not Applicable in Flexbox)

Note: justify-items is a CSS Grid property, not Flexbox. In Grid, it aligns grid items along the inline (row) axis within their grid areas. Flexbox uses justify-content instead for similar functionality.

For comparison, in Grid: justify-items: center would center items horizontally within their grid cells.

Flexbox equivalent: Use justify-content on the container to align items along the main axis.

align-self (Individual Item Alignment)

Allows individual flex items to override the container's align-items value. Applied directly to flex items, not the container.

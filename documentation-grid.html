Flex Start CSS Grid System Documentation

This page explains the responsive 12-column grid system built into flex-start.css. It includes default columns plus breakpoint variants .sm-*, .md-* and .lg-*.

Grid container and base column

Use .grid to create a row container. Use .col for equal-width fill or .col-{n} for fixed fractions.

.grid
display: flex; flex-wrap: wrap;
.col
flex: 1; default fill
.col-1 ... .col-12
1/12 to 12/12 width

Responsive Breakpoints

The system supports mobile-first classes using .sm-*, .md-*, and .lg-*.

.sm-1 ... .sm-12
Breakpoints <= 767px
.md-1 ... .md-12
Breakpoints 768px - 991px
.lg-1 ... .lg-12
Breakpoints >= 1200px

Column rules and notes

  • One row can hold up to 12 columns. When columns exceed 12, they wrap to the next line.
  • Never combine grid and flex helper classes on the same element, except when intentionally nested.
  • Use .col for equal fluid columns, and .col-{n} for exact fractions.
  • Breakpoint classes are mobile-first and override defaults at their screen width range.

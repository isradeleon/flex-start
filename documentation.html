Flex Start CSS Documentation

This page documents all utility classes available in flex-start.css, with live examples and short descriptions. It uses the same design style as demo.html to keep things consistent.

Core Flex Container Classes

Use these classes to initialize a flexbox container.

.flex
display: flex
.flex.inline
display: inline-flex
.flex.wrap
flex-wrap: wrap
.flex.wrap-reverse
flex-wrap: wrap-reverse
.flex.reverse
flex-direction: row-reverse
.flex.column
flex-direction: column
.flex.column.reverse
flex-direction: column-reverse

Quick example:

.flex.wrap.justify-between.items-center

Other Notes

  • Use .flex first to establish the flex container, then add alignment modifiers.
  • .flex.content-* only affects wrapped multi-line containers.
  • Use .self-* on child elements only.
  • .grow-* and .shrink-* work per-item, not container-wide.
